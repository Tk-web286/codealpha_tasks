# Phishing Awareness Training Portal

## Overview

The Phishing Awareness Training Portal is an interactive cybersecurity awareness website developed as part of the CodeAlpha Cyber Security Internship. The project aims to educate users about phishing attacks, fake websites, social engineering tactics, and cybersecurity best practices through engaging and interactive learning modules.

This portal helps users identify common phishing techniques and provides practical guidance to prevent becoming a victim of cyberattacks.

---

## Features

### 1. Introduction to Phishing
- Overview of phishing attacks
- Importance of cybersecurity awareness
- Common attack methods used by cybercriminals

### 2. Types of Phishing
- Email Phishing
- Spear Phishing
- Whaling
- Smishing (SMS Phishing)
- Vishing (Voice Phishing)

### 3. Interactive Phishing Email Analysis
- Simulated phishing email examples
- Identification of suspicious sender addresses
- Detection of malicious links and urgent requests

### 4. Fake Website Detection
- Comparison between legitimate and fraudulent websites
- URL verification techniques
- Website authenticity checks

### 5. Social Engineering Awareness
- Fear-based attacks
- Urgency tactics
- Authority impersonation
- Curiosity exploitation
- Reward and giveaway scams

### 6. Cybersecurity Best Practices
- Verify sender information
- Inspect URLs before clicking
- Enable Multi-Factor Authentication (MFA)
- Use strong passwords
- Keep software updated
- Report suspicious communications

### 7. Interactive Quiz
- Multiple-choice cybersecurity questions
- Automated scoring system
- User knowledge assessment

### 8. Certificate Generation
- Personalized certificate upon successful completion
- Training completion verification
- User achievement recognition

### 9. Mobile Responsive Design
- Responsive layout for all devices
- Hamburger navigation menu
- Enhanced user experience

### 10. Cybersecurity Theme
- Modern cyber-inspired user interface
- Dark mode design
- Interactive and engaging visuals
